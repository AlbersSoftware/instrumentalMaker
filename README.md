# Wave-U-net-TF2
This repository implements the [Wave-U-net](https://github.com/f90/Wave-U-Net) architecture in TensorFlow 2

# Usage Example

```
git clone https://github.com/satvik-venkatesh/Wave-U-net-TF2.git
cd Wave-U-net-TF2

from wave_u_net import wave_u_net

m = wave_u_net()

m.summary()
```
